# 检索问题库

用于把每次研究从“随机搜索”转成稳定问题清单。运行周报、月报、季度复盘或竞品档案时，应根据任务选择对应问题。

## 通用检索问题

1. 本周期市场需求是否变化：新能源汽车保有量、家充/公共桩规模、充电量、政策是否有新信号？
2. 同规格产品价格是否下降，RFQ 目标价是否下移？
3. 客户是否增加供应商或压低目标成本？
4. 标准、CCC、GB/T、OCPP、海外认证是否变化？
5. 是否出现新的招投标、中标、采购公告或车企定点？
6. 是否出现我方核心客户、目标客户或竞品客户重叠？
7. 公开资料能否支撑结论；哪些必须转为内部数据缺口？

## 挚达/智达检索问题

1. 最近是否新增主机厂客户或进入我方核心客户？
2. 是否披露新的出货量、收入、毛利率、亏损、现金流？
3. 是否披露安装服务覆盖城市、服务次数、SLA 或数字平台能力？
4. 是否涉及五菱、小鹏或我方目标客户？
5. 是否出现低价中标、疑似低价压力或价格锚下移？
6. 低价是否有回收路径：安装、售后、后市场、海外、资本市场？
7. 是否从“同维低价竞争”升级为“主机厂配套 + 安装服务 + 平台 + 资本”的多维竞争？

## 科大智能检索问题

1. 最新年报/季报中数字能源收入、毛利、现金流、应收账款是否变化？
2. 充电桩业务是否有明确收入、产能、客户或订单披露？
3. 参股/子公司是否进入比亚迪、吉利、蔚来、长城、小米、五菱、小鹏等车企？
4. 是否有 AC 家充、随车配交流桩、快充直流、超充产品变化？
5. 是否出现低价中标或改变目标客户价格锚？
6. 其电网/储能/配电能力是否构成我方难以复制的系统方案优势？

## 公牛检索问题

1. 公牛新能源汽车充电桩线上排名、销量和价格是否变化？
2. 线下网点、安装服务、售后承诺是否增强？
3. 是否进入车企、4S 店、官方附件或后市场渠道？
4. 是否推出低价家充产品或扩展功率段？
5. 我方是否有车企同源后市场侧翼机会，还是应继续避免 C 端正面战？

## 星星充电检索问题

1. 公共充电运营规模、充电量、排名是否变化？
2. 目的地充电、4S、园区或小功率 DC 场景是否增强？
3. 是否通过站点资源或平台资源控制利润池？
4. 我方小功率 DC 是否应继续试点，还是需要升级优先级？

## Wattsaving/能效电气检索问题

1. 20/30/40/60/80kW 小功率 DC 产品是否新增型号、认证或客户？
2. 是否有可验证订单、连续 RFQ 或中标？
3. 场景是否可复制：4S、园区、目的地、海外样板？
4. 毛利、安装、售后和认证路径是否能闭环？

## Tesla China 检索问题

1. 家充服务、认证安装商、目的地充电政策是否变化？
2. 是否影响车企对家充/安装服务的期望？
3. 是否改变 OEM 官方附件、后市场或安装服务标准？
