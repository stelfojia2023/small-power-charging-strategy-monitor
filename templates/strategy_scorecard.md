## 评分说明

分值范围：0-5。分数越高，代表对当前战略的调整压力越大。每个维度必须给出证据等级和内部数据缺口。

| 维度 | 分值 | 说明 | 本期证据 | 证据等级 | 内部数据缺口 |
|---|---:|---|---|---|---|
| 市场需求变化 |  | 需求增强或减弱，对现有产品线的影响 |  |  |  |
| 商品化压力 |  | 价格下降、供应商增加、RFQ 压价 |  |  |  |
| 竞品威胁 |  | 挚达/科大/公牛等是否加压 |  |  |  |
| 利润池变化 |  | 利润是否转向服务、渠道、安装 |  |  |  |
| 我方可复制性 |  | 对手打法我方是否能复制 |  |  |  |
| 负毛利风险 |  | 是否触发止损 |  |  |  |
| 研发资源影响 |  | 是否需要调整投入 |  |  |  |

## 总体判断规则

| 总体判断 | 条件 |
|---|---|
| 维持战略 | 无高风险项，主要假设成立，核心维度不超过 3 分 |
| 小幅调整 | 1-2 个维度超过 3 分，或某个非核心假设部分变化 |
| 重大调整 | 2 个以上核心假设部分失效，或客户/利润池/竞品结构发生实质变化 |
| 立即止损 | 负毛利风险、客户集中恶化、无回收路径同时出现 |
