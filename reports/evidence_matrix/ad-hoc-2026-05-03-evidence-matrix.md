# 小功率充电桩战略证据矩阵

刷新日期：2026-05-03  
适用报告：`reports/monthly/monthly-2026-05-03-small-power-charging-strategy-review.md`

## 1. 执行结论

- <span style="color:orange">公开证据支持：家充/私人充电设施仍是最大需求底盘，OEM/ODM 方向继续成立。</span>
- <span style="color:red">公开证据支持：挚达/智达是复合型竞品，威胁来自 OEM、安装服务、海外和平台，不只是低价硬件。</span>
- <span style="color:red">内部必需：五菱/小鹏低价、负毛利、价格锚变化不能由公开资料闭环，必须补报价、BOM、项目毛利和服务范围。</span>
- <span style="color:red">强证据约束：电动汽车供电设备 CCC 认证在2026年8月1日成为出厂、销售、进口和经营使用硬门槛。</span>

## 4. 证据等级说明

- `强`：政府、监管、标准、交易所、年报、招股书、正式招投标。
- `中`：公司官网、车企官网、权威媒体、协会数据。
- `弱`：自媒体、论坛、截图、经销商说法。
- `内部必需`：公开资料无法闭环，必须补报价、BOM、毛利、RFQ、安装、返修等内部数据。
- `证据不足`：不能支持结论，只能继续跟踪。
