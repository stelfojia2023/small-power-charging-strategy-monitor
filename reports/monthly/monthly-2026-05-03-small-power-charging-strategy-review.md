## 6. 市场动态

### 国内市场

截至2026年3月底，国家能源局口径显示全国充电基础设施2148.1万个，私人充电设施1661.8万个，私人设施约为公共设施的3.4倍。这个结构说明家庭、随车、私人车位和官方家充包仍是小功率 AC/便携产品的核心需求底盘。

但整车端没有给供应链放松空间。中汽协公开数据表明，2026年一季度新能源汽车累计销量296万辆，同比下降3.7%；国内市场承压而出口高增长。对我方的含义是：国内 OEM 项目会继续压价，出口型车企和海外配套可能成为更好的增量入口。

<span style="color:orange">结论：市场不是没有需求，而是利润池从标准硬件转向成本、认证、服务、渠道和出口配套。</span>

### 海外市场

海外 AC 机会不应泛化为“出海零售”。更合理的路径是绑定车企出口、海外认证、ODM 供货和区域售后伙伴。挚达海外收入同比增长约70.5%，说明竞品已经把海外作为增长和利润恢复路径之一。

<span style="color:green">结论：海外 AC 应作为选择性 ODM 和认证能力建设方向，不能按 C 端品牌出海打法投入。</span>

### 标准与认证

市场监管总局和国家认监委已经明确电动汽车供电设备 CCC 认证节奏：2025年3月1日起受理，2026年8月1日起未获得 CCC 认证证书和标注认证标志的产品不得出厂、销售、进口或经营使用。

<span style="color:red">结论：CCC 已经从合规事项变成产品线准入门槛。便携枪、AC、DC 的研发排期必须把认证样机、差距整改、测试资源和证书转换纳入硬计划。</span>

## 8. 重点竞品深度分析

### 挚达/智达

1. 它依靠什么赢：车企客户、家充产品、安装服务网络、海外增长和平台化运营。
2. 它控制了什么稀缺资源：OEM 入口和全国范围安装服务组织能力。
3. 低价是否有回收路径：公开资料不能证明单项目低于成本，但其可能通过安装服务、海外业务、规模交付和资本市场叙事恢复利润。
4. 我方能否复制：不能复制全国安装网络和资本打法；能复制的是平台化 VAVE、质量交付、区域伙伴和车企响应速度。
5. 对五菱/小鹏影响：若进入同项目 RFQ 或改变目标价，应升级为 P0；当前仍缺内部报价和项目证据。
6. 应对：<span style="color:orange">侧翼防守，不正面复制。用 TCO、质量、交付和认证能力守住高质量订单，负毛利订单必须有恢复路径。</span>

### 公牛

公牛控制的是 C 端信任、终端网点、电商排名和安装售后触达。它不是我方短期应该正面进攻的对象。对我方更合理的打法是借车企同源背书做后市场产品，而不是独立砸品牌和渠道。

### 科大智能

科大智能的风险不是已经证明低价，而是其数字能源、电网/储能/工程方案能力可能在部分客户场景中形成系统优势。公开资料不足以证明其进入我方核心客户或低于成本报价，必须转销售 RFQ 和客户项目证据跟踪。

## 12. 6-24个月战略

- OEM客户扩展：从五菱、小鹏扩到5-8家，优先出口型车企、官方家充包和后市场附件场景。

- 后市场：只做车企同源产品和区域试点，不做全国独立 C 端品牌战。
- 区域安装：建立伙伴池和 SLA，不自建重资产全国网络。
- 海外 AC：优先 ODM、认证和车企出口配套，避免无渠道品牌出海。
- 小功率 DC：20/30kW 保留样机、认证预研和明确客户牵引，不做全线铺开。
- <span style="color:red">暂停方向：无订单的小功率 DC 全系列、全国安装重资产网络、泛 C 端 App、没有恢复路径的负毛利订单。</span>

## 15. 最终决策

1. <span style="color:orange">现在是否需要调整战略？</span>

   <span style="color:orange">需要小幅调整，决策等级为 B。不是放弃 OEM/ODM，而是提高接单纪律、认证优先级和客户分散速度。</span>

2. <span style="color:orange">应该调整什么？</span>

   <span style="color:orange">把执行重点从“拿 OEM 订单”调整为“有毛利红线地拿 OEM 订单”；把降本从项目响应调整为平台化 VAVE；把后市场从观察调整为车企同源小试点；把认证从质量事务调整为产品线准入工程。</span>

3. <span style="color:red">必须立即做什么？</span>

   <span style="color:red">五菱/小鹏项目红线复核、挚达报价和服务包对标、CCC认证倒排、新 OEM 客户漏斗、VAVE 降本清单。</span>

4. <span style="color:red">不应该做什么？</span>

   <span style="color:red">不打公牛式 C 端品牌战；不复制挚达全国安装网络；不把小功率 DC 升级为全线主战场；不接受没有恢复路径的负毛利订单；不在 CCC 未闭环前扩散过多新型号。</span>

5. <span style="color:gray">下次应该跟踪什么？</span>

   <span style="color:gray">五菱/小鹏最新 RFQ、挚达同项目报价和服务范围、公牛新能源渠道变化、科大智能是否进入目标 OEM、小功率 DC 是否出现连续订单、CCC 认证进度和海外 AC 认证需求。</span>
