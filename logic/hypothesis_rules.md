# H1-H6 假设规则

本文件承接 `strategy-monitor/inputs/strategic_assumptions.md` 和 `strategy-monitor/inputs/strategic_thresholds.md`，用于把证据转成稳定判断。

| 假设 | 主要判断 |
|---|---|
| H1 商品化 | 价格下降、供应商增加、BOM透明、毛利下降时成立 |
| H2 OEM/ODM 基本盘 | 车企仍有家充包、随车充、出口配套或官方附件需求时成立 |
| H3 挚达复合竞品 | 同时出现 OEM、安装服务、平台、海外、资本或核心客户重叠时强化 |
| H4 避免公牛 C 端正面战 | 公牛渠道/品牌/服务增强且我方缺少稀缺资源时成立 |
| H5 小功率 DC 试点 | 无连续订单、毛利和服务闭环时维持订单驱动 |
| H6 客户集中和负毛利 | 核心客户集中、价格锚下移、项目毛利低于红线时高风险 |
