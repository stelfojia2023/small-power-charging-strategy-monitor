# 运行流程

标准流程：

```text
读取输入 -> 读取 sources/cache/data/logic -> 判断证据时效 -> 刷新缺失或关键证据 -> 更新证据库 -> 生成报告 -> 刷新 viewer
```

周期定位：

| 周期 | 重点 |
|---|---|
| 周报 | 高风险信号、竞品动作、政策/标准快扫 |
| 月报 | H1-H6 假设验证、证据矩阵、行动项 |
| 季报 | 历史趋势、证据等级变化、未闭环缺口 |
| 半年报 | 战略方向、资源配置、客户结构 |
| 年报 | 全年复盘、战略假设重设、下一年目标 |

每次新增可复用证据后，同步更新 `cache/evidence_index.md`。修改 Markdown 后运行 `python3 tools/build_viewer_data.py`。
