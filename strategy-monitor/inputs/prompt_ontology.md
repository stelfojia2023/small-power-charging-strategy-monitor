# 提示词本体与别名

用于统一检索词、报告名称和竞品称呼。

## 产品与功率段

| 标准名称 | 别名/相关词 | 监控意图 |
|---|---|---|
| 便携充电枪 | 随车充、随车桩、便携式充电器、portable EV charger | 判断商品化、OEM 配套、价格压力 |
| 家用 AC 充电桩 | 家充、私人桩、wallbox、AC charger | 判断家充需求、安装服务、后市场 |
| AC 3.5kW | 3.5kW、低功率家充 | 判断低价车型配套 |
| AC 7kW | 7kW、单相家充 | 判断主流家充平台 |
| AC 11kW | 11kW、三相家充 | 判断海外和高端车型适配 |
| AC 21kW | 21kW、三相高功率 AC | 判断高端/海外场景 |
| 小功率 DC | 小直流、小功率直流、20/30/40/60/80kW DC | 判断目的地、4S、园区和补能场景 |

## 商业模式与工具词

| 标准名称 | 别名/相关词 | 监控意图 |
|---|---|---|
| OEM/ODM | 主机厂配套、整车厂供货、车企供应链 | 判断基本盘和客户拓展 |
| RFQ | 询价、招标、采购、项目报价 | 判断价格锚和客户机会 |
| VAVE | 降本、价值工程、平台化、替代料 | 判断成本优势 |
| TCO | 全生命周期成本、安装售后成本、质量成本 | 反制低价报价 |
| 车企同源后市场 | 4S 后市场、车企商城、官方附件 | 判断轻渠道机会 |
| 区域安装伙伴 | 安装服务商、认证安装商、区域服务网络 | 判断服务能力补齐 |

## 竞品名称

| 标准名称 | 别名/相关词 | 注意事项 |
|---|---|---|
| 挚达/智达 | Zhida、挚达科技、智达 | 报告中可写“挚达/智达”，避免不同来源命名混乱 |
| 科大智能 | CSG Smart Science、300222 | 重点看数字能源、车企客户和充电相关业务 |
| 公牛 | 公牛集团、Bull、603195 | 重点看 C 端渠道、品牌和安装服务 |
| 星星充电 | Star Charge、万帮星星充电 | 重点看运营网络、目的地和场景资源 |
| Wattsaving | 能效电气、小功率 DC 厂商 | 重点看 20/30/40kW DC 产品和海外场景 |
| Tesla China | 特斯拉中国 | 重点看家充、认证安装商、目的地充电 |

## 海外地区

海外 AC 不得作为一个整体判断，至少拆分：

- 欧盟：AFIR、CE、IEC、OCPP、公共补能规则。
- 英国：UKCA、智能充电、建筑法规 Part S。
- 北美：UL、NACS/J3400、NEC、DOE 数据。
- 东南亚：本地认证、插头标准、车企出口和渠道伙伴。
