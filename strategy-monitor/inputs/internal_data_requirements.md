# 内部数据需求清单

公开资料无法闭环的判断，必须显式索取内部数据。

## 客户与项目

| 字段 | 用途 | 责任建议 |
|---|---|---|
| 客户名称 | 判断客户集中度和核心客户风险 | 销售 |
| 项目名称/车型/平台 | 判断产品适配和生命周期 | 销售/项目 |
| RFQ 阶段 | 判断机会真实性 | 销售 |
| 目标价/客户压价幅度 | 判断价格锚变化 | 销售 |
| 我方报价 | 判断毛利和竞争力 | 销售/财务 |
| 竞品报价 | 判断低价压力 | 销售 |
| 交付范围 | 判断报价是否可比 | 项目/售后 |
| 安装/售后责任 | 判断 TCO 和服务成本 | 售后 |

## 成本与毛利

| 字段 | 用途 | 责任建议 |
|---|---|---|
| BOM 成本 | 判断 VAVE 空间和负毛利风险 | 研发/采购 |
| 制造成本 | 判断真实交付成本 | 制造/财务 |
| 认证成本 | 判断产品线投入 | 研发/质量 |
| 物流成本 | 判断跨区域/出口毛利 | 供应链 |
| 安装成本 | 判断服务利润池 | 售后 |
| 返修率/故障率 | 判断质量 TCO | 质量/售后 |
| 单项目毛利率 | 判断接单红线 | 财务 |
| 单客户毛利率 | 判断客户集中质量 | 财务 |

## VAVE 与平台化

| 字段 | 用途 | 责任建议 |
|---|---|---|
| 共用件比例 | 判断平台化程度 | 研发 |
| 替代料清单 | 判断短期降本 | 研发/采购 |
| 关键器件价格趋势 | 判断成本风险 | 采购 |
| 认证复用性 | 判断平台扩展效率 | 质量/认证 |
| 功率段复用边界 | 判断 AC/DC 产品路线 | 研发 |

## 输出要求

报告中遇到内部数据缺失时，必须填写：

```text
缺失字段：
影响的判断：
当前只能得出的结论：
责任建议：
补数截止时间：
```
