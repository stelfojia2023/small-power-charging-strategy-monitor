# 挚达/智达多维竞争风险模型

挚达/智达是当前最高优先级竞品之一。不得只把它理解为低价硬件供应商，必须同时验证主机厂配套、规模交付、安装服务、平台数据、资本支撑、海外增长和后市场入口。

## 风险维度

| 维度 | 要验证的问题 | 关键证据 | 对我方含义 |
|---|---|---|---|
| 主机厂规模交付 | 是否新增车企客户，是否进入五菱、小鹏或我方目标客户 | 招股书、公告、车企采购、RFQ、定点 | 可能直接重叠我方 OEM 基本盘 |
| 价格锚攻击 | 是否用低价重设五菱/小鹏价格体系 | 内部竞品报价、客户目标价、招投标价格 | 影响毛利红线和接单策略 |
| 安装服务回收 | 是否通过安装、售后、服务包补回硬件低毛利 | 服务覆盖、安装次数、SLA、收费模式 | 说明其低价可能不是纯亏损 |
| 平台与数据 | 是否增强数字化服务平台、用户运营和车主触达 | 官网、招股书、平台功能、APP、车主服务 | 可能控制后市场入口 |
| 资本支撑 | 亏损是否可被融资、上市资金或资本故事支撑 | 融资、招股书现金流、募资用途 | 解释短期低价可持续性 |
| 海外增长 | 是否用海外更高毛利补贴国内低价 | 海外收入、客户、认证、渠道 | 国内低价压力可能延长 |
| 后市场入口 | 是否通过车企客户进入车主补购、换新、安装服务 | 官方附件、4S 后市场、车主商城 | 改变利润池位置 |
| 对我方影响 | 是否从同维低价竞争变成多维竞争 | 上述证据综合 | 决定维持、小幅调整、重大调整或止损 |

## 判断规则

| 风险等级 | 条件 | 应对 |
|---|---|---|
| 观察 | 只有宣传、媒体转述或单点产品信息 | 继续跟踪，不调整战略 |
| 中 | 有车企客户或安装服务证据，但未进入我方核心客户 | 做产品/成本/服务对标 |
| 高 | 进入我方核心客户 RFQ、定点或改变价格锚 | 启动 P0 报价、BOM、服务范围和毛利核查 |
| 极高 | 低价、客户重叠、服务回收、资本支撑同时出现 | 升级管理层决策，评估止损或客户结构调整 |

## 必填输出

每次涉及挚达/智达时，报告必须回答：

```text
它依靠什么赢？
它控制了什么稀缺资源？
低价是否有回收路径？
我方能否复制？
如果不能复制，应正面进攻、侧翼进攻、合作，还是避开？
对五菱/小鹏/新 OEM 客户有什么直接影响？
```
